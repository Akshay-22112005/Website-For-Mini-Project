Personal AI Chatbot is a smart, multilingual virtual assistant designed to provide meaningful, real-time conversations across various topics such as health, fitness, motivation, finance, travel, and education.

Built with Flutter (frontend) and Flask (backend) and powered by Googleâ€™s Gemini AI, this application combines intelligent natural language understanding with a sleek, interactive user interface.

Key Features

  • Conversational AI: Generates intelligent, human-like responses.
  • Multilingual Support: English, Hindi, and Kannada.
  • Dynamic Topics: Quick buttons for instant chats.
  • Interactive Interface: Light/Dark theme, smooth animations.
  • Persistent Context: Keeps replies relevant.
  • Auto Reset: Refreshes idle chats.
  • Integrated Animations: Lottie effects enhance UI polish.

Tech Stack

  • Frontend: Flutter (Dart)
  • Backend: Flask (Python)
  • AI Engine: Google Gemini API
  • State Management: Provider
  • Networking: HTTP requests
  • Animations: Lottie

How It Works

  1. User sends a message or selects a topic.
  2. The message is sent via HTTP POST to Flask backend.
  3. Flask forwards it to Gemini AI with language info.
  4. Gemini generates a response and sends it back.
  5. The Flutter app displays it elegantly.

Vision

This project merges AI, Flutter, and Flask to create a smart, multilingual assistant that adapts to user needs and enhances productivity through real-time conversations.
